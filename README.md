
# snap

Small helper for [sharefile](https://github.com/jb55/sharefile)

  1. Takes a screen shot with `scrot`, passing args

  2. Sends screenshot to sharefile

  3. Copies link to clipboard

## Installing

    $ nix-env -f https://jb55.com/pkgs -iA snap

## Dependencies

* libnotify
* scrot
* sharefile
* xclip
